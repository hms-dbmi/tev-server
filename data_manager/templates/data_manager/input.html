File Format

Tab-delimited files with these five fields may be uploaded:

Required Fields:

  • Subject ID
  • Day
  • Gene
  • AA_change
  • VAF

The first row of the file should label the columns, and the labels must match exactly. Several optional fields are supported:

Optional Fields

  • Chr
  • Position
  • Total Read
  • Ref_read
  • Var_read
  • Alt
  • Ref
  • cDNA_change
  • VariantType
  • Assay
  • RefSeq

For additional information on these parameters, a page explaining each field can be found here.
